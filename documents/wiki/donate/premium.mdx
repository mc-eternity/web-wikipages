# PREMIUM

Выдается навсегда, наследует команды статуса [VIP](https://www.etty.su/docs/wiki/donate/vip), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Например, если вы являетесь HELPER (Администрация) и имеете статус PREMIUM (привилегия), вы будете обладать всеми возможностями доната PREMIUM наряду с функциями HELPER.

## Команды

| Команда | Описание |
| :---------: | :---------: |
| /kit premium | Кит PREMIUM (каждые 15 дней) |
| /options | Отключить тп к себе |
| /seen ник | Узнать, когда игрок был в сети |
| /anvil | Открыть наковальню |
| /ctable | Стол картографа |
| /ci | Очистить инвентарь |
| /stable | Стол кузнеца |
| /sc | Камнерез |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе```
- ```25``` слотов на аукционе, ```8``` точек дома
- ```10``` регионов привата по ```500.000``` блоков
- ```5``` работ в ```/jobs```
- Флаг ```ПЛОД-ХОРУСА``` в ```/flags```
- Автовыдача денег раз в ```30``` минут ```30$```

## Как получить

Данную привилегию можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```1,490 фунтов``` (донат-валюта).
