# GHOST

Выдается навсегда, наследует команды статуса [SHADOW](https://www.etty.su/docs/wiki/donate/shadow), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус GHOST (привилегия), вы будете обладать всеми возможностями доната GHOST наряду с функциями HELPER.

## КОМАНДЫ

## Команды

| Команда | Описание |
| :---------: | :---------: |
| /kit ghost | Кит GHOST (каждые 15 дней) |
| /rg setpriority [рг] [число] | Приоритет в рг |
| /feed | Накормить себя |
| /near | Найти игроков рядом |
| /dback | Вернуться на место смерти |
| /notarget | Стать незаметным для мобов |
| /sun /rain /storm | Изменить погоду в мире |
| /warp set | Создать свой варп |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе```
- ```35``` слотов на аукционе, ```12``` точек дома
- ```20``` регионов привата по ```700.000``` блоков
- ```8``` работ в ```/jobs```
- Флаг ```ПОДНЯТИЕ ПРЕДМЕТОВ``` в ```/flags```
- Автовыдача денег раз в ```30``` минут ```50$```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```4,390 фунтов``` (донат-валюта).
