# Сезонный пропуск 
- Сезонный пропуск представляет собой более ```100``` уникальных заданий, которые растянуты на весь сезон. 

<img src="https://raw.githubusercontent.com/mc-eternity/web-wikipages/refs/heads/master/assets/sp_info.webp" />

--------

## Основная информация
- Сезонный пропуск включает в себя разные типы заданий: ```добыча блоков, крафт предметов, убийство мобов, повышение уровня работ и скиллов, сдача предметов и интерактивные квесты.```
- За выполнение данных заданий можно получить различные награды: ```доллары, кейсы, фунты```.

--------

## Основные команды 

| Команда | Описание |
| :---------: | :---------: |
| /seasonpass | Открывает сезонный пропуск |
| /sp | Открывает сезонный пропуск |

--------

## Пропуск заданий
- Вы можете пропустить нынешнее задание без получения награды.
- Цена пропуска задания - ```250 фунтов```.
