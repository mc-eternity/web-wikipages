# PlasmoVoice (голосовой чат)

**PlasmoVoice** - модификация для Minecraft, которая позволяет общаться на сервере с помощью **голосового чата**. Данная информация также доступна в игре при открытии меню помощи по голосовому чату.

--------

## Помощь по установке

1. **Перейти по ссылке** — нажмите **ЛКМ** на иконку монитора в меню помощи в игре, чтобы получить ссылку на загрузку мода.

2. **Выбрать версию на сайте:**
   - Поддерживаются версии от **1.19.4** до **1.21.11+**
   - Укажите **модификатор**: **Fabric** или **Forge**

3. **Установить мод в папку игры:**
   - Откройте проводник, в строке поиска меню «Пуск» введите **%appdata%** и откройте результат
   - Перейдите в папку **.minecraft**, затем в папку **mods**
   - Скопируйте скачанный файл мода в папку **mods**

**Ссылка на загрузку:** [Modrinth — PlasmoVoice](https://modrinth.com/plugin/plasmo-voice/versions)

--------

## Помощь по запуску

После установки мода запустите игру через лаунчер в соответствующей версии:

**Версия:** 1.19.4 – 1.21.11+ (Fabric или Forge). Точная версия указана в названии файла мода.

В игре нажмите кнопку **V**, чтобы открыть настройки PlasmoVoice:
- **Микрофон** — выбор устройства записи
- **Устройства ввода** — настройка устройств
- **Активация голосового чата** — по **кнопке** или по **голосу**

--------

## Создание групп (каналов)

В меню PlasmoVoice (по умолчанию кнопка **V**) откройте вкладку **Активации** и настройте **Группы** (Groups).

| Действие                  | Команда                                              |
|---------------------------|------------------------------------------------------|
| Создать публичную группу  | `/groups create название`                            |
| Создать группу с паролем  | `/groups create name: название password: пароль`     |
| Список общедоступных групп| `/groups` или `/groups browse`                       |

## macOS

Если **Legacy Launcher не видит микрофон** в macOS, выполните по шагам:

1. **Закройте лаунчер и игру** (если они запущены).

2. **Сбросьте разрешения микрофона:** откройте **Терминал** и выполните команду:
   ```tccutil reset Microphone```

3. **Настройте лаунчер:**
   - Откройте лаунчер → **Настройки** → **Лаунчер**
   - В пункте **«При запуске Minecraft»** включите **«Закрывать Legacy Launcher»**
   - Сохраните настройки

4. **Проверка в игре:**
   - Запустите Minecraft с установленным модом и зайдите на сервер
   - В настройках PlasmoVoice выберите **активацию по голосу**
   - Включите значок микрофона и поочерёдно переключайте **устройства ввода** (микрофоны), пока нужное не начнёт работать, запросив разрешение (выдайте разрешение)

Так же может потребоваться закрыть принудительно лаунчер! Он должен быть закрыт обязательно.

## Кастомные пластинки

На сервере поддерживаются кастомные музыкальные пластинки — вы можете загрузить любой трек по ссылке и записать его на обычную пластинку.

| Команда                          | Описание                                   |
|----------------------------------|--------------------------------------------|
| `/disc burn <ссылка> [название]` | Создаёт пластинку с кастомным аудио по URL |
| `/disc cancel`                   | Прерывает процесс поиска или записи        |
| `/disc erase`                    | Возвращает кастомную пластинку в обычную   |
| `/disc search <запрос>`          | Ищет треки по названию                     |
