# INFINITY

Выдается навсегда, наследует команды статуса [ULTIMATE](https://etty.site/docs/wiki-skyblock/donate/ultimate), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус INFINITY (привилегия), вы будете обладать всеми возможностями доната INFINITY наряду с функциями HELPER.

## Команды

| Команда       | Описание                      |
|---------------|-------------------------------|
| /kit infinity | Кит INFINITY (каждые 14 дней) |
| /fly          | Бесплатный полёт везде        |
| /god          | Режим бессмертия              |
| /vanish       | Режим невидимки               |
| /walkspeed    | Изменить скорость ходьбы      |
| /prefix       | Изменить цвет префикса        |
| /feed all     | Накормить всех                |
| /heal all     | Вылечить всех                 |
| /dhomes       | Дополнительные точки домов    |
| /tempban      | Забанить игрока (макс 7 дней) |
| /tempmute     | Замутить игрока (макс 7 дней) |
| /warn         | Выдать варн игроку            |
| /kick         | Кикнуть игрока                |
| /inspect      | История блока                 |

!!! Наказания можно выдавать только по правилам. !!!

## Дополнительные возможности

- Префикс в ```чате``` и ```табе``` (ранг **INFINITY**)
- Уникальный остров в ```/is```
- Возможность скролла картин через **СКМ**
- ```80``` слотов на аукционе, ```30``` точек дома
- ```6``` варпов доступно для создания
- ```100``` предметов в ```/ft```
- ```15``` предметов в ```/autosell```
- ```10``` миньонов к постановке в ```/mech```
- ```10``` воронок к постановке в ```/mech```
- ```10``` печей к постановке в ```/mech```
- ```10``` голограмм для создания
- Автовыдача денег раз в ```30``` минут ```100$```
- Возможности предыдущего доната, кроме ```/kit```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```64,490 рубинов``` (донат-валюта).
