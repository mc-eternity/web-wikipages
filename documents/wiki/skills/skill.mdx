# Скиллы

> Развивайте свои способности, используя систему скиллов.

## Что дают скиллы

- Увеличение уровня персонажа
- Ускорение добычи ресурсов
- Увеличение здоровья и выносливости
- Другие игровые бонусы

## Как развивать скиллы

1. Введите команду ```/skills``` для открытия меню
2. Выберите интересующий вас скилл
3. Выполняйте соответствующие задачи для получения опыта

## Как повышать уровни

1. Зарабатывайте очки опыта, выполняя действия связанные со скиллом
2. Используйте накопленные очки для повышения уровня

## Важные заметки

- Каждый скилл имеет максимальный уровень
- Выбирайте скиллы, соответствующие вашему стилю игры

**Совет:** Сфокусируйтесь на развитии 2-3 основных скиллов для лучшей эффективности.
