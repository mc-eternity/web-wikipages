# LEGEND

Выдается навсегда, наследует команды статуса [SANURAI](https://www.etty.su/docs/wiki/donate/samurai), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус LEGEND (привилегия), вы будете обладать всеми возможностями доната LEGEND наряду с функциями HELPER.

## Команды

| Команда               | Описание                                |
|-----------------------|-----------------------------------------|
| /kit legend           | Выдача набора "legend" (каждые 15 дней) |
| /fly                  | Бесплатный полёт                        |
| /flyspeed ник         | Изменить скорость полёта игрока "ник"   |
| /heal ник             | Вылечить игрока "ник"                   |
| /repair all           | Починить все вещи в инвентаре           |
| /bc                   | Сделать объявление для всех             |
| /glow                 | Включить подсветку персонажа            |
| /back                 | Вернуться в последнее сохраненное место |
| /mirror (start\|stop) | Зеркалирование постройки (start|stop)   |


## Дополнительные возможности

- Префикс в ```чате``` и ```табе```
- Выпадает спавнер с шансом - ```10%```
- Писать ```жирным``` текстом в чате - ```/colors```
- ```60``` слотов на аукционе, ```20``` точек дома
- ```50``` регионов привата по ```2.500.000``` блоков
- ```4``` варпа доступно для создания
- Флаг ```ВХОД В РЕГИОН``` в ```/flags```
- Создать лифт - ```[ЛИФТ]``` на табличке
- Автовыдача денег раз в ```30``` минут ```80$```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```15,990 фунтов``` (донат-валюта).
