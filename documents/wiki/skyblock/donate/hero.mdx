# HERO

Выдается навсегда, наследует команды статуса [PLAYER](https://etty.site/docs/wiki-skyblock/donate/player), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус HERO (привилегия), вы будете обладать всеми возможностями доната HERO наряду с функциями HELPER.

## Команды

| Команда     | Описание                      |
|-------------|-------------------------------|
| /kit hero   | Кит HERO (каждые 14 дней)     |
| /loom       | Ткацкий станок                |
| /craft      | Открыть верстак               |
| /grindstone | Точильный камень              |
| /hat        | Надеть предмет на голову      |
| /ignore     | Игнорировать игрока           |
| /join       | Сообщение о входе             |
| /silence    | Скрыть сообщения игроков      |
| /spin       | Крутиться                     |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе``` (ранг **HERO**)
- Уникальный остров в ```/is```
- ```20``` слотов на аукционе, ```5``` точек дома
- ```3``` активных квеста в ```/quests```
- ```6``` предметов в ```/ft```
- Автовыдача денег раз в ```30``` минут ```20$```
- Возможности, доступные для игрока — ```/help```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```990 рубинов``` (донат-валюта).
