# SHADOW

Выдается навсегда, наследует команды статуса [PREMIUM](https://etty.site/docs/wiki-survival/donate/premium), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус SHADOW (привилегия), вы будете обладать всеми возможностями доната SHADOW наряду с функциями HELPER.

## Команды

| Команда      | Описание                    |
|--------------|-----------------------------|
| /kit shadow  | Кит SHADOW (каждые 15 дней) |
| /ptime       | Установить личное время     |
| /pweather    | Установить личную погоду    |
| /ext         | Потушить себя               |
| /ec          | Эндер-сундук                |
| /clan create | Создать свой клан           |
| /options     | Отключить приём денег       |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе```
- ```30``` слотов на аукционе, ```10``` точек дома
- ```15``` регионов привата по ```600.000``` блоков
- ```6``` работ в ```/jobs```
- Флаг ```ВЗАИМОДЕЙСТВИЕ``` в ```/flags```
- Автовыдача денег раз в ```30``` минут ```40$```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```3,000 фунтов``` (донат-валюта).
