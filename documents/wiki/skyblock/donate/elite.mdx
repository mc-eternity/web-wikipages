# ELITE

Выдается навсегда, наследует команды статуса [DELUXE](https://etty.site/docs/wiki-skyblock/donate/deluxe), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус ELITE (привилегия), вы будете обладать всеми возможностями доната ELITE наряду с функциями HELPER.

## Команды

| Команда       | Описание                     |
|---------------|------------------------------|
| /kit elite    | Кит ELITE (каждые 12 дней)   |
| /heal         | Вылечить себя                |
| /repair       | Починить вещь в руке         |
| /expfly сумма | Полёт за плату опытом        |
| /flyc         | Выключить временный полёт    |
| /itemname     | Дать название предмету       |
| /itemlore     | Дать описание предмету       |
| /day /night   | Изменить время в мире        |
| /chatcolor    | Цвет сообщения в чате        |
| /colorpicker  | Выбор hex цвета              |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе``` (ранг **ELITE**)
- Уникальный остров в ```/is```
- Писать ```цветным``` текстом в чате — ```/colors```
- ```40``` слотов на аукционе, ```14``` точек дома
- ```5``` активных квестов в ```/quests```
- ```20``` предметов в ```/ft```
- ```6``` миньонов к постановке в ```/mech```
- ```6``` воронок к постановке в ```/mech```
- ```6``` печей к постановке в ```/mech```
- ```2``` варпа доступно для создания
- Автовыдача денег раз в ```30``` минут ```60$```
- Возможности предыдущего доната, кроме ```/kit```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```11,499 рубинов``` (донат-валюта).
