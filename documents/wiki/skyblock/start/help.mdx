# Особенности сервера

Руководство по основным возможностям сервера. Данная информация также доступна в игре при введении команды ```/help```

--------

## Система тикетов

Используйте тикеты для сообщения о проблемах или вопросе администрации сервера.

### Как создать тикет:
1. Введите команду:  ```/ticket <причина>```  
*(Ваше сообщение отправится в очередь к администраторам)*

**Пример использования:**  
```/ticket игрок devetty нарушил п.1.2 правил```

### Важные примечания:
1. 🕒 Администрация обрабатывает тикеты **по очереди**
2. ✍️ Всегда указывайте **максимум деталей** о проблеме
3. ⚠️ Наказание предусмотрено за:
- Бессмысленные тикеты
- Злоупотребление системой

**Рекомендации:**
- Используйте четкие формулировки
- Укажите координаты места проблемы (если актуально)
- Не создавайте дублирующие тикеты

## Управление островом

### Команды

- ```/is``` — общее меню острова
- ```/is invite <ник>``` — добавить игрока на остров
- ```/is help``` — общий список островных команд

### При удалении острова

- Очищается инвентарь, эндер-сундук.
- Деньги с баланса банка возвращаются вам.

### Следует помнить

Создание нового острова — затратная операция, поэтому игрок может пересоздать остров **5** раз за вайп.

## Вайп сервера

Периодически на сервере проводится вайп (полная очистка данных).

### Что пропадает:
- Наигранная статистика: баланс, постройки, убийства, смерти, б.пропуск, уровни скиллов, работы и др.

### Что остается:
- Привилегии
- Косметика
- Рубины

## Игровой Чат

### [G] Глобальный чат
- **Префикс:** `!` в начале сообщения.  
- **Действие:** Сообщение видно на всем сервере.  

### [L] Локальный чат
- **Префикс:** Не требуется. 
- **Действие:** Сообщение видно в радиусе 100 блоков.  

### [T] Торговый чат
- **Префикс:** `$` в начале сообщения.  
- **Действие:** Сообщение видно на всем сервере.  
- **Задержка:** 10 минут = 1 сообщение

### [ЛС] Личные сообщения
- **Команды:**  
- ```/m [ник] [сообщение]``` (Пример: ```/m devetty привет```).  
- ```/msg [ник] [сообщение]```.  
- **Быстрый ответ:** `/r [сообщение]` (ответит последнему отправителю).

## Создание пузырьков опыта

### Основной способ:
1. Возьмите **пустую бутылочку** в руку
2. Нажмите `ПКМ` по столу зачарования

### Массовое создание:
- Для быстрого создания **10 пузырьков** одновременно:
`Shift + ПКМ`  по столу зачарования

**Примечание:**
- Пузырьки опыта используются для хранения и переноса опыта
- Стол зачарования должен быть доступен для взаимодействия

## Аукцион предметов

Покупайте и продавайте предметы на аукционе с другими игроками.

### Продажа предметов
1. Возьмите предмет в руку
2. Введите команду:   ```/ah sell <цена>```
3. Выберите валюту для продажи из предложенных вариантов

### Покупка предметов
1. Откройте аукцион: (```/ah```)
2. Нажмите `ЛКМ` на понравившийся предмет
3. Подтвердите покупку

### Поиск предметов
- **По категориям**:
1. Откройте аукцион (```/ah```)
2. Выберите нужную категорию в левом меню

- **По игроку**:
 ```/ah search <ник>```  
 *(покажет все лоты указанного игрока)*

- **По предмету**:
  ```/ah search <название или id предмета>```  
  *(покажет все лоты совпадающие с запросом)*

## Использование лифта

Создавайте и используйте лифты для перемещения между этажами, в том числе когда этажей больше 2.  

**Доступно с донат-статуса:** [ULTIMATE](https://etty.site/docs/wiki-skyblock/donate/ultimate)

### Как создать лифт 
1. Установите табличку на каждом этаже.  
2. Напишите на ней: `[лифт]` (одинаково сверху и снизу).  

### Как использовать лифт 
- **Подняться вверх:** `ПКМ`  
- **Спуститься вниз:** `Shift + ПКМ`  

**Примечание:**  

Лифт работает только если таблички находятся на одинаковых координатах **X и Z** (проверьте через **F3**).

## Создание варпов

Создавайте и посещайте варпы других игроков для торговли, путешествий и иных целей.  

**Доступно с донат-статуса:** [DELUXE](https://etty.site/docs/wiki-skyblock/donate/deluxe)  

### Как создать варп

1. Найдите подходящее место.  
2. Введите команду:  ```/warp set <название>```  

*Варп будет создан в вашем текущем местоположении.*  

### Управление варпами

- **Открыть меню:**  
  ```/warps```  
- **Удалить варп:**  
  ```/warp remove <название>```  
- **Добавить описание:**  
  ```/warp desc set <название> <описание>```  
- **Оценить варп (1–5):**  
  ```/warp rate <название> <оценка>```

## Передача доната

Пользователи, получившие привилегию ниже своей (путём покупки на [сайте](https://etty.site/) или выигрыша из [Кейса с Донатом](https://etty.site/docs/wiki-survival/cases/donate)), могут передать её другому игроку или продать.

**Доступно с донат-статуса:** [PHANTOM](https://etty.site/docs/wiki-skyblock/donate/phantom)

### Команды

- ```/rewards``` – открыть хранилище донатов  
- ```/giftdonate <ник> <донат>``` – подарить донат 

### Пример использования

```/giftdonate devetty hero``` – подарит игроку **devetty** донат **[HERO](https://etty.site/docs/wiki-skyblock/donate/hero)**.

## Быстрый крафт бетона

Превращайте сухой бетон в бетон с помощью котла.

**Доступно с донат-статуса:** [ULTIMATE](https://etty.site/docs/wiki-skyblock/donate/ultimate)

### Как это работает

1. Поставьте и наполните котёл водой.
2. Держите в руке сухой бетон.
3. Бросьте в котёл сухой бетон (можно стак).
4. Через несколько секунд он превратится в обычный бетон.

### Особенности

1. Необходимо постоянно пополнять котёл водой.
2. Можно за раз использовать только стак.

### Примечание

1. Данный метод позволяет быстрее производить бетон из сухого бетона.
2. Отключить данную функцию можно командой:  
  ```/tconcrete```
