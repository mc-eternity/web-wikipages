# SHINOBI

Выдается навсегда, наследует команды статуса [GHOST](https://etty.site/docs/wiki/donate/ghost), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус SHINOBI (привилегия), вы будете обладать всеми возможностями доната SHINOBI наряду с функциями HELPER.

## Команды

| Команда       | Описание                     |
|---------------|------------------------------|
| /kit shinobi  | Кит SHINOBI (каждые 15 дней) |
| /heal         | Вылечить себя                |
| /repair       | Починить вещь в руке         |
| /expfly сумма | Полёт за плату опытом        |
| /flyc         | Выключить временный полёт    |
| /itemname     | Дать название предмету       |
| /itemlore     | Дать описание предмету       |
| /day /night   | Изменить время в мире        |
| /compass      | Компас, указывающий на цель  |
| /chatcolor    | Цвет сообщения в чате        |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе```
- Писать ```цветным``` текстом в чате - ```/colors```
- ```40``` слотов на аукционе, ```14``` точек дома
- ```25``` регионов привата по ```800.000``` блоков
- ```10``` работ в ```/jobs```
- ```2``` варпа доступно для создания
- Флаг ```УРОН ОТ ПАДЕНИЯ``` в ```/flags```
- Автовыдача денег раз в ```30``` минут ```60$```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```10,000 фунтов``` (донат-валюта).
