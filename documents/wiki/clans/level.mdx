## Как повысить уровень 

- Проходя клановые задания, вы зарабатываете очки клана, которые помогают повысить уровень. Кроме того, в заданиях могут быть различные предметы.
- Задания могут быть самыми разнообразными: например, на разрушение блоков, на рыбалку, на переплавку, на поедание, на зачарование, на убийство мобов, на получение опыта, а также многоуровневые задания.
- Так же есть другие варианты.

## Очки клана
- Очки клана можно как получить, так и потерять.

- Получение клановых очков:

| Действие | Очки | 
| :---------: | :---------: |
| Выполнение квеста | до +500 |
| Новый участник | +20 |
| Убийство игрока | +2 |

- Потеря клановых очков:

| Действие | Очки | 
| :---------: | :---------: |
| Смерть участника | -2 |
| Выход из клана | -500 |
| Кик из клана | -1000 |
