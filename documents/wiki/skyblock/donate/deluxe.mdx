# DELUXE

Выдается навсегда, наследует команды статуса [PRIME](https://etty.site/docs/wiki-skyblock/donate/skyblock-prime), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус DELUXE (привилегия), вы будете обладать всеми возможностями доната DELUXE наряду с функциями HELPER.

## Команды

| Команда           | Описание                      |
|-------------------|-------------------------------|
| /kit deluxe       | Кит DELUXE (каждые 14 дней)   |
| /feed             | Накормить себя                |
| /near             | Найти игроков рядом           |
| /dback            | Вернуться на место смерти     |
| /notarget         | Стать незаметным для мобов    |
| /sun /rain /storm | Изменить погоду в мире        |
| /warp set         | Создать свой варп             |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе``` (ранг **DELUXE**)
- Уникальный остров в ```/is```
- ```35``` слотов на аукционе, ```12``` точек дома
- ```15``` предметов в ```/ft```
- ```8``` предметов в ```/autosell```
- ```5``` миньонов к постановке в ```/mech```
- ```5``` воронок к постановке в ```/mech```
- ```5``` печей к постановке в ```/mech```
- ```1``` варп доступен для создания
- Автовыдача денег раз в ```30``` минут ```50$```
- Возможности предыдущего доната, кроме ```/kit```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```4,839 рубинов``` (донат-валюта).
