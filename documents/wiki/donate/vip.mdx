# VIP

Выдается навсегда, наследует команды статуса [PLAYER](https://www.etty.su/docs/wiki/donate/player), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Например, если вы являетесь HELPER (Администрация) и имеете статус VIP (привилегия), вы будете обладать всеми возможностями доната VIP наряду с функциями HELPER.

## Команды

| Команда | Описание |
| :---------: | :---------: |
| /kit vip | Кит VIP (каждые 15 дней) |
| /loom | Ткацкий станок |
| /craft | Открыть верстак |
| /grindstone | Точильный камень |
| /hat | Надеть предмет на голову |
| /ignore | Игнорировать игрока |
| /join | Сообщение о входе |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе```
- ```20``` слотов на аукционе, ```5``` точек дома
- ```5``` регионов привата по ```400.000``` блоков
- ```4``` работы в ```/jobs```
- Флаг ```ЭНДЕР-ЖЕМЧУГ``` в ```/flags```
- Автовыдача денег раз в ```30``` минут ```20$```
- Команды доступные для игрока - ```/help```

## Как получить

Данную привилегию можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```990 фунтов``` (донат-валюта).
