# Материалы для крафта
> Уникальные материалы для крафта улучшенных инструментов с особыми ванильными зачарованиями

--------

## Общая информация

Оба материала:
- Не могут быть созданы, только получены
- Используются в специальных рецептах крафта

--------

## Осколок измерения

<img src="https://raw.githubusercontent.com/mc-eternity/web-wikipages/refs/heads/master/assets/oskolok.webp" />

**Свойства**:
- Используется для крафта уникальных инструментов

**Источники получения**:
1. Награды сезонного пропуска
2. Кейсы:
   - Обычный (низкий шанс)
   - Плюс (средний шанс)
   - Редкий (высокий шанс)
