# Боевой Пропуск

Боевой Пропуск — это система заданий с наградами за выполнение.

<img src="https://s3.etty.site/web/wiki/survival/battlepass/info/menu.webp" />

## Что важно знать

Доступны два типа пропуска:

- Обычный - доступен всем.

- [Премиум](https://etty.site/docs/wiki-survival/battlepass/bpinfo#Премиум%20пропуск) - с дополнительными наградами и квестами.

Статус можно посмотреть в меню пропуска ```/bp``` или в скорборде (справа на экране).

## Основные команды

| Команда | Описание |
| :---------: | :---------: |
| /bp | Открыть меню Боевого Пропуска |
| /bpq | Открыть задания (меню заданий) |
| /bpr | Посмотреть награды |
| /bp booster | Посмотреть активные бустеры Боеого пропуска |
| /bp help | Показать список всех доступных команд |
| /bp leaderboard | Лидеры по количеству выполненных заданий |
| /bp stats | Посмотреть личную статистику |

## Премиум пропуск

<img src="https://s3.etty.site/web/wiki/survival/battlepass/info/premium.webp" />

Премиум-пропуск открывает:

- Повышенные награды.

- Эксклюзивные задания.

- Ускоренный прогресс.

Купить можно в [магазине](https://etty.site/).
