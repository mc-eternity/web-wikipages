# PHANTOM

Выдается навсегда, наследует команды статуса [ELITE](https://etty.site/docs/wiki-skyblock/donate/elite), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус PHANTOM (привилегия), вы будете обладать всеми возможностями доната PHANTOM наряду с функциями HELPER.

## Команды

| Команда       | Описание                              |
|---------------|---------------------------------------|
| /kit phantom  | Кит PHANTOM (каждые 12 дней)          |
| /mfly сумма   | Полёт за деньги                       |
| /flyc         | Выключить временный полёт             |
| /ph           | Создание голограмм на острове         |
| /ec ник       | Посмотреть чужой эндер-сундук         |
| /feed ник     | Накормить другого игрока              |
| /invsee ник   | Посмотреть чужой инвентарь            |
| /rewards      | Выдать донат игроку из кейса          |
| /emojipicker  | Выбор эмодзи для чата                 |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе``` (ранг **PHANTOM**)
- Уникальный остров в ```/is```
- Писать ```курсивным``` текстом в чате — ```/colors```
- Писать цветным текстом на табличках
- ```45``` слотов на аукционе, ```16``` точек дома
- ```25``` предметов в ```/ft```
- ```12``` предметов в ```/autosell```
- ```7``` миньонов к постановке в ```/mech```
- ```7``` воронок к постановке в ```/mech```
- ```7``` печей к постановке в ```/mech```
- ```3``` варпа доступно для создания
- ```3``` голограммы для создания
- Автовыдача денег раз в ```30``` минут ```70$```
- Возможности предыдущего доната, кроме ```/kit```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```7,199 рубинов``` (донат-валюта).
