# SAMURAI

Выдается навсегда, наследует команды статуса [SHINOBI](https://www.etty.su/docs/wiki/donate/shinobi), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Например, если вы являетесь HELPER (Администрация) и имеете статус SAMURAI (привилегия), вы будете обладать всеми возможностями доната SAMURAI наряду с функциями HELPER.

## Команды

| Команда | Описание |
| :---------: | :---------: |
| /kit samurai | Кит SAMURAI (каждые 15 дней) |
| /mfly сумма | Полёт за деньги |
| /flyc | Выключить временный полёт |
| /ec ник | Посмотреть чужой эндер сундук |
| /feed ник | Накормить другого игрока |
| /invsee ник | Посмотреть чужой инвентарь |
| /rewards | Выдать донат игроку из кейса |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе```
- Написание ```курсивным``` текстом в чате - ```/colors```
- ```45``` слотов на аукционе, ```16``` точек дома
- ```35``` регионов привата по ```1.500.000``` блоков
- ```3``` варпа доступно для создания
- Флаг ```РАЗМЕЩЕНИЕ ТРАНСПОРТА``` в ```/flags```
- Автовыдача денег раз в ```30``` минут ```70$```

## Как получить

Данную привилегию можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```5,990 фунтов``` (донат-валюта).
