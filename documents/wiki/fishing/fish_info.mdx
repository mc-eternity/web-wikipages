# Кастомная рыбалка

На сервере доступна уникальная система рыбалки, которую при желании можно отключить! 

--------

## Особенности:  

Вместо стандартных рыб вы будете ловить кастомных — с разными названиями, ценами, редкостью и весом.

Пойманную добычу можно продать скупщику через команду – `/fish shop`!  

--------

## Все команды:  

| Команда         | Описание                                               |
|-----------------|--------------------------------------------------------|
| `/fish help`    | Показать эту страницу помощи                           |
| `/fish gui`     | Открыть главное меню                                   |
| `/fish next`    | Показать, сколько времени осталось до следующего соревнования |
| `/fish shop`    | Открыть магазин для продажи рыбы                       |
| `/fish sellall` | Продать всю рыбу в инвентаре                           |
| `/fish toggle`  | Включить или отключить получение особой рыбы           |
| `/fish top`     | Показать таблицу лидеров текущего соревнования         |

--------

## Соревнования и призы

В системе рыбалки проводятся соревнования, в которых можно получить призы за попадание в топ:

| Место   | Приз       |
|---------|------------|
| 1 место | 15 фунтов  |
| 2 место | 10 фунтов  |
| 3 место | 5 фунтов   |
