# Материалы для крафта

## Осколок измерения

**Свойства**:
- Используется для крафта уникальных инструментов

**Источники получения**:
1. Награды сезонного пропуска
2. Кейсы:
   - Обычный (низкий шанс)
   - Плюс (средний шанс)
   - Редкий (высокий шанс)
