# КЛАН

Клан — это объединение игроков для совместной игры.

--------

## Условия создания

Для создания клана необходимо выполнить следующие условия:

1.  На вашем игровом балансе должно быть не менее:  
   **`10.000 $`**

2. Вы должны иметь привилегию **`SHADOW`** (или выше)

--------

## Как сделать название клана цветным?

Для начала нужно создать клан с любым названием:

1.  На вашем игровом балансе должно быть не менее:  
   **`10.000 $`**

2. Вы должны иметь привилегию **`SHADOW`** (или выше)

3. ```/clan create <название>```

После создания клана нужно переименовать его:

1. [Сайт для подбора градиента](https://www.birdflop.com/resources/rgb/?colors=%5B%7B%22hex%22%3A%22%23F5C987%22%2C%22pos%22%3A0%7D%2C%7B%22hex%22%3A%22%23FCD05C%22%2C%22pos%22%3A100%7D%5D&shadowcolors=%5B%7B%22hex%22%3A%22%233D3222%22%2C%22pos%22%3A0%7D%2C%7B%22hex%22%3A%22%233F3417%22%2C%22pos%22%3A100%7D%5D&text=ETERNITY&format=%7B%22color%22%3A%22MiniMessage%22%2C%22bold%22%3A%22%3Cb%3E%24t%3C%2Fb%3E%22%2C%22italic%22%3A%22%3Ci%3E%24t%3C%2Fi%3E%22%2C%22underline%22%3A%22%3Cu%3E%24t%3C%2Fu%3E%22%2C%22strikethrough%22%3A%22%3Cst%3E%24t%3C%2Fst%3E%22%2C%22obfuscate%22%3A%22%3Cobf%3E%24t%3C%2Fobf%3E%22%7D)
