# ULTIMATE

Выдается навсегда, наследует команды статуса [PHANTOM](https://etty.site/docs/wiki-skyblock/donate/skyblock-phantom), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус ULTIMATE (привилегия), вы будете обладать всеми возможностями доната ULTIMATE наряду с функциями HELPER.

## Команды

| Команда               | Описание                                      |
|-----------------------|-----------------------------------------------|
| /kit ultimate         | Кит ULTIMATE (каждые 14 дней)                 |
| /is fly               | Бесплатный полёт на своём острове             |
| /flyspeed             | Изменить скорость полёта                      |
| /heal ник             | Вылечить игрока                               |
| /repair all           | Починить все вещи в инвентаре                 |
| /bc                   | Сделать объявление для всех                 |
| /nf                   | Снять с себя эффекты                          |
| /glow                 | Включить подсветку персонажа                  |
| /back                 | Вернуться в последнее сохранённое место       |
| /tconcrete            | Крафт бетона через котёл                      |
| /height 0.6-1.3       | Изменить размер персонажа                     |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе``` (ранг **ULTIMATE**)
- Уникальный остров в ```/is```
- Писать ```жирным``` текстом в чате — ```/colors```
- ```60``` слотов на аукционе, ```20``` точек дома
- ```4``` варпа доступно для создания
- ```6``` активных квестов в ```/quests```
- ```50``` предметов в ```/ft```
- ```8``` миньонов к постановке в ```/mech```
- ```8``` воронок к постановке в ```/mech```
- ```8``` печей к постановке в ```/mech```
- ```5``` голограмм для создания
- Лифт: табличка с ```[ЛИФТ]```
- Автовыдача денег раз в ```30``` минут ```80$```
- Возможности предыдущего доната, кроме ```/kit```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```31,300 рубинов``` (донат-валюта).
