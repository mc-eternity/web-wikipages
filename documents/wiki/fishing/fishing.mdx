# Кастомная рыбалка

> На сервере есть кастомная  
> и отключаемая рыбалка!  

--------

## Особенности:  

> Вместо обычных рыб, вам выпадает  
> кастомная, которую можно продать  
> скупщику рыбы – `/fish shop`!  

--------

## Все команды:  

| Команда      | Описание                      |
| :----------: | :---------------------------: |
| `/fish next` | Следующее соревнование        |
| `/fish shop` | Скупщик рыбы                  |
| `/fish sellal` | Продать всю рыбу             |
| `/fish toggle` | Вкл/выкл режим кастомной рыбалки             |
| `/fish top`  | Топы соревнования             |
