# EMPEROR

Выдается навсегда, наследует команды статуса [LEGEND](https://www.etty.su/docs/wiki/donate/legend), кроме ```/kit```. Не является привилегией Администрации, но может быть совместно использована с ней.

Если вы являетесь HELPER (Администрация) и имеете статус EMPEROR (привилегия), вы будете обладать всеми возможностями доната EMPEROR наряду с функциями HELPER.

## Команды

| Команда | Описание |
| :---------: | :---------: |
| /kit emperor | Кит EMPEROR (каждые 20 дней) |
| /god | Режим бессмертия |
| /vanish | Режим невидимки |
| /walkspeed | Изменить скорость ходьбы |
| /prefix | Изменить цвет префикса |
| /feed all | Накормить всех |
| /heal all | Вылечить всех |
| /clear | Очистить инвентарь |
| /tempban | Забанить игрока (макс 7 дней) |
| /tempmute | Замутить игрока (макс 7 дней) |
| /warn | Выдать варн игроку |
| /kick | Кикнуть игрока |

## Дополнительные возможности

- Префикс в ```чате``` и ```табе```
- Выпадает спавнер с шансом - ```15%```
- ```80``` слотов на аукционе, ```30``` точек дома
- ```75``` регионов привата по ```3.750.000``` блоков
- ```6``` варпов доступно для создания
- Флаг ```СПАВН МОБОВ``` в ```/flags```
- Автовыдача денег раз в ```30``` минут ```100$```

## Как получить

Можно получить путем прямой покупки с сайта, с помощью доплаты или купить у других игроков, в случаях, когда она им выпала из кейса и храниться в хранилище наград ```/rewards```.

**Цена доплаты:** ```50,490 фунтов``` (донат-валюта).
